Welcome to my website

Madlax is an object based operating system that is being developed for use in PC's and embedded systems. It has been designed to work in almost any software and hardware environment and can be reconfigured to suit the requirements of the target system, as well as individual users. The user interface is completely rewriteable and is capable of cloud-based operation.

Madlax is currently available for x86-based and risc-based machines for free. The commercial version is in development right now - check back in future for information on its availability.

Madlax's features include:

A Completely Rewritable Graphical Interface. 100% Component Based/Plug-In Architecture. Portable to Most Platforms and Hardware Devices. Supports DML Scripting. Audio support for WAV, MOD, M4A, MP3, MP4, MIDI, AAC, WMA, AIFF and others. Graphical support for PNG, JPEG, JP2000, HPGS, EXR, PCX, IFF, WEBP, TIFF, BMP, GIF and others. Video support for MPEG2, MPEG4, DIVX3, WMV, MOV, AVI, DVD, BLURAY and others.
Integrated ZIP compression. Hardware accelerated opengl/es based graphics architecture. OpenCL support built-in. Full Support for Unicode (International Character Sets). Utilises Load on Demand Processing for Quick Load Times and Minimal Memory Usage. Based Entirely on Modern Technology - No Legacy Code. Free version available for download. Source Code is Publicly Available.
