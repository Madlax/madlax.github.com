Madlax

The object based operating system

Welcome to my website

Madlax is an object based operating system that is being developed for use in PC's and embedded systems. It has been designed to work in almost any software and hardware environment and can be reconfigured to suit the requirements of the target system, as well as individual users. The user interface is based upon the award winning GNOME and Enlightenment desktop environments.

Madlax is in development right now - check back in the future for information on its progress.

Madlax's features include:

  • A Completely Scriptable Graphical Interface.
  • 100% Component Based/Plug-In Architecture.
  • Portable to Most Platforms and Hardware Devices.
  • Supports Window Scripting.
  • Audio support for WAV, MOD, M4A, MP3, MP4, MIDI, AAC, WMA, AIFF and others.
  • Graphical support for PNG, JPEG, WEBP, EXR, TIFF, MJPEG and others.
  • Video support for MPEG4, DIVX, WMV, MOV, AVI, DVD, BLURAY and others.
  • Integrated AirPlay support.
  • Integrated ZIP compression.
  • Hardware accelerated OpenGL based gaming architecture.
  • OpenAL.
  • Flash based user interface.
  • Sharing of media libraries between multiple devices on your home network.
  • Sharing and playing of multimedia streams over bluetooth.
  • OpenCL support built-in.
  • Full Support for Unicode (International Character Sets).
  • Load on Demand Processing for Quick Load Times and Minimal Memory Usage.
  • Based Entirely on Modern Technology - No Legacy Code.
  • Source Code is Publicly Available.
  • Hardware drivers for all major graphics and audio cards.
